# MemoBlock 全屏阅读模式设计

## 概述

为 MemoBlockV3 添加全屏阅读按钮，用户点击后进入纯阅读模式，提供沉浸式阅读体验。

## 需求

- 点击全屏按钮进入阅读模式（模态框）
- 移动端和桌面端行为一致
- 提供明显的退出按钮
- 阅读模式下隐藏所有操作按钮

## UI 设计

### 布局结构

```
┌─────────────────────────────────────────┐
│  ✕                                        │  ← 顶部退出按钮 (固定)
│─────────────────────────────────────────│
│                                           │
│         Markdown 内容                       │
│         (居中，最大宽度 800px)             │
│                                           │
└─────────────────────────────────────────┘
```

### 视觉设计

- **退出按钮位置**: 右上角，固定定位
- **内容区域**: 居中显示，最大宽度 800px
- **背景**: 深色半透明背景 (#000/50%)
- **动画**: fade-in + scale-in 过渡

### 组件状态

| 状态 | 显示内容 |
|------|----------|
| 正常 | MemoBlockV3 原样显示 |
| 阅读模式 | 模态框 + 内容 + 退出按钮 |

## 实现要点

### 1. 状态管理

```tsx
const [isReadingMode, setIsReadingMode] = useState(false);
```

### 2. 按钮位置

- 桌面端 (sm+): Footer 工具栏，Copy 和 Share 旁边
- 移动端: Footer 工具栏保留（因为空间充足）

### 3. 全屏按钮图标

使用 lucide-react 的 `Maximize2` 图标

### 4. 退出按钮

- 图标: `X` 或 `Minimize2`
- 位置: 右上角固定
- 点击关闭模态框
