## 战略判断

### 做/不做/差异化

- [ ] **做** - 符合差异化优势 + 用户需求 + 我们优势
- [ ] **不做** - 不符合本地化定位 / 跟随无意义 / 我们做没有优势
- [ ] **差异化** - 同样的用户价值，用我们独特的方式实现

### 优先级

- [ ] **P0** - 核心差异化（利用本地/单二进制/Geek Mode 优势）
- [ ] **P1** - 重要增强（用户强需求）
- [ ] **P2** - 未来考虑（有价值但非当前重点）
