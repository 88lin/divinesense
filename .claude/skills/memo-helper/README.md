# Memo Helper - 笔记助手

快速将内容记录到 DivineSense 服务器（**自动登录**）。

## 快速开始

```
/memo 今天完成了新功能的开发
```

## 自动登录

**配置文件**: `~/.memo-config.json`

```json
{
  "server": "http://39.105.209.49",
  "username": "你的用户名",
  "password": "你的密码"
}
```

**优先级**: 配置文件 > 环境变量 > 询问用户

## 命令

| 命令    | 描述               |
| :------ | :----------------- |
| `/memo <内容>` | 快速记录笔记（自动登录）|

## 环境变量

| 变量 | 说明 |
|:-----|:-----|
| `MEMO_SERVER` | 服务器地址 |
| `MEMO_USERNAME` | 用户名 |
| `MEMO_PASSWORD` | 密码 |

## 特性

- ✅ **自动登录**: 配置一次，永久有效
- ✅ **Token 缓存**: 自动缓存，减少登录次数
- ✅ **标签提取**: 自动识别 `#标签`
- ✅ **Markdown 支持**: 保留格式
