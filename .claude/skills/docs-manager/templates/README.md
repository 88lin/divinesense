# 文档管理技能模板

本目录包含 `docs-manager` 技能使用的各种模板。

---

## 模板文件

### dev-guide-template.md

开发指南模板，用于创建新的开发指南文档。

### research-report-template.md

研究报告模板，用于创建新的调研文档。

### spec-template.md

规格文档模板，用于创建新的实施规格。

### archive-readme-template.md

归档目录 README 模板。

---

## 使用方法

```bash
# 复制模板到目标位置
cp templates/spec-template.md docs/specs/phase-2/team-a/P2-A001-new-feature.md

# 或通过 /docs-new 命令自动使用模板
/docs-new spec new-feature --phase=2 --team=a
```
